## Typography

### Primary Font: Inter

**Usage:** Headings, UI, marketing materials

**Weights:**
- Regular (400) - Body text
- Medium (500) - Subheadings
- Semibold (600) - Headings
- Bold (700) - Emphasis

**Why Inter:**
- Modern, clean, professional
- Excellent readability
- Open source (SIL Open Font License)
- Great for digital and print

---

### Code Font: JetBrains Mono

**Usage:** Code examples, technical documentation

**Weights:**
- Regular (400) - Code blocks
- Bold (700) - Emphasis in code

**Why JetBrains Mono:**
- Designed for developers
- Excellent ligature support
- Clear character distinction
- Open source (Apache 2.0)

---

**Sans-serif:** -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif

**Monospace:** "JetBrains Mono", "Fira Code", "Courier New", monospace

## Usage Guidelines

### Logo Usage

**Do:**
- Use official logo files
- Maintain aspect ratio
- Provide clear space (minimum 20px)
- Use on appropriate backgrounds
- Scale proportionally

**Don't:**
- Distort or stretch
- Change colors (except approved variations)
- Add effects (shadows, gradients)
- Rotate or skew
- Place on busy backgrounds

---

### Color Usage

**Do:**
- Use primary colors for main elements
- Use neutral colors for text and backgrounds
- Use semantic colors for states
- Maintain sufficient contrast (WCAG AA)
- Test on different backgrounds

**Don't:**
- Use too many colors at once
- Use colors that clash
- Ignore accessibility guidelines
- Create custom brand colors

---

### Typography Usage

**Do:**
- Use specified fonts
- Maintain hierarchy
- Ensure readability
- Use appropriate weights
- Consider line length

**Don't:**
- Mix too many fonts
- Use decorative fonts
- Ignore hierarchy
- Use tiny font sizes
- Create walls of text

## Accessibility

### Color Contrast

All color combinations meet WCAG AA standards:
- Text: 4.5:1 minimum
- Large text: 3:1 minimum
- UI components: 3:1 minimum

### Alternative Text

All images include:
- Descriptive alt text
- Context for screen readers
- Meaningful descriptions

### Responsive Design

All assets work at:
- Mobile (320px+)
- Tablet (768px+)
- Desktop (1024px+)
- Large screens (1920px+)
